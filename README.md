# SistemaPersonajesVideojuego

Trabajo elaborado por Aythami Reyes, Alejandro Acosta, Damián Pérez.
Descripcion breve.

Proyecto para desarrollar la lógica de un sistema de personajes de videojuego, con sus atributos, los personajes comparten características comunes pero cada personaje actúa de forma distinta(ataque, habilidades, capacidades). En la jerarquía MVC, tenemos una clase abstranta personaje la cual es obligatoria, que a su vez tiene otras dos subclases personajefisico y personajemagico, con al menos algun metodo no abstracto como por ejemplo cargarAtaque. Mímino 3 interfaces obligatorias, clase gestora del juego, Main para ejecutar y mostrar, y consola para imprimir y leer.
